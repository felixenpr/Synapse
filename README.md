# Synapse - Tarjeta de Presentación Digital

Una tarjeta de presentación digital moderna construida con Next.js 16, React 19 y Tailwind CSS 4.

##  Características

-  **Diseño Moderno**: UI elegante con tema claro/oscuro automático
-  **Rendimiento Optimizado**: Compilado con Turbopack, imágenes optimizadas
-  **Responsive**: Perfectamente adaptado para dispositivos móviles
-  **Sistema de Diseño**: Variables CSS avanzadas
-  **Seguridad**: Headers de seguridad configurados
-  **Temas**: Modo claro/oscuro automático

##  Stack Tecnológico

- **Framework**: Next.js 16.0.6
- **Runtime**: React 19.2.0
- **Styling**: Tailwind CSS 4.1.9
- **Icons**: Lucide React 0.454.0
- **Temas**: Next Themes
- **Analytics**: Vercel Analytics

##  Instalación

\\\ash
pnpm install
cp .env.example .env.local
\\\

##  Desarrollo

\\\ash
pnpm dev
\\\

Abre http://localhost:3000
